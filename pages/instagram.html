Adding an Instagram feed to your page

Adding an Instagram feed to any webpage you've created in FreshPlate is as easy as 1,2,3.

1) Start by adding these styles to your own CSS:

#instaFeed li { list-style-type: none; display: inline-block; margin: .75em; } #instaFeed img { padding: .25em; border: solid 1px #ccc; }

You will not need the following header h1 style if you are using the optional class "reverseTransparent" as in the html below:

#instaFeed header h1 { text-align: center; }

2) Paste the code below into your page where you'd like to display the Instagram feed.

<section class="fineHead reverseTransparent"> <!-- the classes applied to the section are optional --> <header> <h1>Instagram Title</h1> </header> <article> <ul id="instaFeed"> </ul> </article> </section>

3) Obtain an Instagram user id and access token. Place them in the jQuery script snippet below after you've pasted it into your page. Since the user id is numeric, not a string, it doesn't need quotes:

<script> $(document).ready(function(){ var token = 'PLACE YOUR INSTAGRAM ACCESS TOKEN HERE BETWEEN THE SINGLE QUOTES', userid = PLACE YOUR INSTAGRAM USER ID HERE WITHOUT QUOTES, numPhotos = 20; $.ajax({ url:'https://api.instagram.com/v1/users/' + userid + '/media/recent/', dataType: 'jsonp', type: 'GET', data: {access_token:token, count:numPhotos}, success:function(data){ console.log(data); for(x in data.data){ $('#instaFeed').append('<li><a href="'+ data.data[x].link +'"><img src="'+ data.data[x].images.thumbnail.url + '"/></a></li>'); } }, error: function(data){ console.log(data); } }) }); // end doc ready </script>

numPhotos is the number of photos from the feed that you want to display; we've set it to 20.

Whether you use reverseTransparent or choose to use no additional classes, you now have a responsive Instagram gallery on your page that will update when your account updates.
